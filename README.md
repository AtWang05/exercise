马原选择题随机刷题系统
项目简介：这是一个专为期末考试、硕士研究生招生考试政治复习打造的马原客观单选题在线练习系统。项目完全基于前端单页应用架构开发，无任何后台依赖，开箱即用，支持手机端与电脑端完美自适应。
在线开始刷题网址：https://atwang05.github.io/exercise/
核心功能
1. 智能随机出题：完整收录 441 道马原经典单项选择题。打破传统文档顺序背题的死板模式，采用纯前端随机抽取算法，真正检验知识点掌握情况。
2. 错题本自动归纳： 答题错误时，系统会自动高亮显示正确答案，并给出清晰的对错反馈。 错题会自动归纳进错题库，方便进行二次针对性高强度复习。
3. 本地持久化存储 (localStorage)：利用浏览器原生缓存技术，即使用户关闭网页、刷新浏览器或手机断网，已积累的错题数据依然存在，拒绝复习进度被意外清空。
技术栈与架构
本系统由 AI 协作工具辅助完成，追求极致的加载速度和零服务器成本：
1. 前端骨架：HTML5 / CSS3（原生 Flex 弹性布局，完美适配全面屏手机与平板）
2. 核心逻辑：原生 JavaScript，不依赖任何重度第三方框架，做到毫秒级首屏渲染。
3. 数据存储：JSON 数据格式硬编码集成 / 浏览器 localStorage
4. 托管平台：GitHub Pages 静态网站托管（零延迟，全网 CDN 加速）
